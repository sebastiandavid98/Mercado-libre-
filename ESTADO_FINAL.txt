🚀 CÓMO COMENZAR (Para cada desarrollador)

PASO 1: Cambiar a tu rama
────────────────────────
$ git checkout feature/portafolio-desarrollador-[TU_NÚMERO]

Opciones:
  - Desarrollador 1: git checkout feature/portafolio-desarrollador-1
  - Desarrollador 2: git checkout feature/portafolio-desarrollador-2
  - Desarrollador 3: git checkout feature/portafolio-desarrollador-3
  - Desarrollador 4: git checkout feature/portafolio-desarrollador-4

PASO 2: Editar tu portafolio
──────────────────────────────
Edita los archivos en:
  📁 portafolios-desarrolladores/desarrollador-[TU_NÚMERO]/

Contenido a incluir (6 commits):
  1. Foto/avatar
  2. Nombre (información personal)
  3. Descripción profesional
  4. Servicios ofrecidos
  5. Tecnologías que maneja
  6. Enlace a GitHub

PASO 3: Hacer commits
──────────────────────
Después de cada sección clave:

$ git add portafolios-desarrolladores/desarrollador-[TU_NÚMERO]/
$ git commit -m "Commit X: [descripción del cambio]"

PASO 4: Hacer push
──────────────────
$ git push origin feature/portafolio-desarrollador-[TU_NÚMERO]

PASO 5: Crear Pull Request
─────────────────────────
- Ir a GitHub/GitLab
- Crear PR desde tu rama feature hacia "develop"
- Esperar aprobación
- Hacer merge
