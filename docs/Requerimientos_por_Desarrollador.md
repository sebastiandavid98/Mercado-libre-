# Requerimientos por Desarrollador

Este documento contiene la sección de portafolio de servicios que debe completar cada integrante del equipo.

Instrucciones:
- Cada desarrollador debe trabajar en su propia rama (por ejemplo: `feature/<tu-nombre>-portafolio`).
- Añade una sección con el título con tu nombre y completa los campos obligatorios listados abajo.
- Haz commits separados y descriptivos por cada cambio que hagas.

Campos requeridos para cada sección:
- Foto o avatar (URL o ruta relativa dentro del repo)
- Nombre
- Descripción profesional (1-3 líneas)
- Servicios ofrecidos (lista)
- Tecnologías que maneja (lista)
- Enlace a GitHub
